# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 3.x.x   | :heavy_check_mark: |
| 2.x.x   | :x:                |
| 1.x.x   | :x:                |

## Reporting a Vulnerability

Open a [new issue](https://github.com/soywod/react-pin-field/issues)
and prefix the title with `[security]`.
